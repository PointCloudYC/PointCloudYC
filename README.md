## Hi there 👋 I'm Chao YIN (尹超)

**Research Scientist · 3D Vision & Geospatial Intelligence**

I work at the intersection of LiDAR-based scene understanding, BIM modeling, and deep learning — with a focus on making 3D point cloud perception practical for the built environment.

### 🔬 Research Interests

- 3D Point Cloud Processing & Segmentation
- Scan-to-BIM / MEP Scene Understanding
- Weakly-supervised & Self-supervised 3D Learning
- Multi-modal Learning for Construction Scenes

### 📦 Open-Source Work

**Datasets**
- [`Industrial3D`](https://github.com/PointCloudYC/Industrial3D) — 620M-terrestrial LiDAR dataset with 12 classsse for industrial MEP scene understanding *an extension of PSNet5 dataset*
- [`PSNet5`](https://github.com/PointCloudYC/ResPointNet2) — 80M-terrestrial LiDAR dataset with 5 classes *(hosted in ResPointNet2)*

**Methods**
- [`ResPointNet2`](https://github.com/PointCloudYC/ResPointNet2) — residual PointNet++ for MEP point cloud segmentation
- [`se-pseudogrid`](https://github.com/PointCloudYC/se-pseudogrid) — SE mechanism applied to PseudoGrid for piping component classification
- [`SQN-pytorch`](https://github.com/PointCloudYC/SQN-pytorch) — 1st PyTorch re-implementation of SQN on GitHub
- [`SQN_tensorflow`](https://github.com/PointCloudYC/SQN_tensorflow) — early TensorFlow re-implementation of SQN (weakly supervised segmentation)

**Research Infrastructure**
- [`Deep-Learning-On-Point-Clouds`](https://github.com/PointCloudYC/Deep-Learning-On-Point-Clouds) — curated reading list & resources
- [`dot-files`](https://github.com/PointCloudYC/dot-files) — dev environment configs
- [`claude-research-skills`](https://github.com/PointCloudYC/claude-research-skills) — LLM-powered academic research automation
